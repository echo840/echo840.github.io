# GitHub Pages Personal Homepage

This repository is a simple static personal homepage for GitHub Pages.

## Files

- `index.html`: homepage entry file
- `assets/css/style.css`: page styles
- `assets/img/avatar.svg`: temporary avatar placeholder
- `jianli.md`: CV/resume placeholder

## Publish

1. Create a GitHub repository named:

   ```text
   your-github-username.github.io
   ```

2. Put these files in the repository root.

3. Push to the `main` branch.

4. In GitHub, open:

   ```text
   Settings -> Pages
   ```

5. Set the source to:

   ```text
   Deploy from a branch
   ```

   Then choose:

   ```text
   main / root
   ```

6. Your site will be available at:

   ```text
   https://your-github-username.github.io/
   ```

## Edit

Replace the placeholder name, email, affiliation, links, publications, and avatar in `index.html`.
